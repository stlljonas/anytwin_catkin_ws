# Author(s): C. Dario Bellicoso            #
############################################

The scripts found in the 'inverse kinematics' folder can be used to generate
the analytical and symbolical inverse kinematics for ANYmal.

To update the symbolic terms used to compute the analytical inverse kinematics,
update the script 'compute3DofIk' and call it to generate the functions.
